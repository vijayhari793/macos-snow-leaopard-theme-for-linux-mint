Objective of this Repo:
This will give the MAC OS SNOW LEOPARD theme for Mint Linux - Cinnamon Release

Procedure :
To install this in linux mint cinnamon first unzip the file then copy the 3 folders to /home/username. 
note to replace username with those folders are in linux mint cinnamon like snow leaopard folder.
actual user name. 
then rename the following apps. no 1 files to Finder. no 2 calendar to iCal. no 3 Disks to Disk utillity. 
to do so
open menu right click the app you want to change then select properties then rename. 
To change the icons of the app you have to
right click the app you want to change icon select properties then click on the icon then click browse then go to 
/home/username/.icontheme 
note to replace username with your actual username. at first the .icontheme may not appear because 
it is hidden if .icontheme folder didnt appear right click at /home/username and select show hidden files then go into the
.icontheme folder and click the app icon example finder finder.jpeg matching icon. there are only few app icos in this version
we may release next version. then there could be more icons. to get the launchpad open system settings you can rename the system
settings to system preferences and use the icons available in the appimages folder for best experience. so first open system 
settings then go to applets then go to downloads tab search for appdrawer after installation go to system settings select applets
then click on appdrawer select add then click on the gear icon then in an option select navigation to scroll horizontal put all background
colour container colour box colour to grey colour and select opacity to 85 percentage. Now go to settings applets download and search
classic menu and download. then go to settings applets then go to classic menu select add then select the gear icon in classic menu
then enable select custom icon and label click on icon select browse then go to /home/username/.icontheme it could be hidden so right
click on empty space and select show hidden files then go to.icons folder select menu.jpeg to get apple like menu then edit panel remove
original menu by right click remove position the new apple menu to the left. then right click on panel select move and position to top
then go to settings themes and select application macos x cheetah master icons to mint x aqua desktop to mint x. then you will get
mac like close options and more skeumorphic ui then open terminal type this command 
gsettings set org.cinnamon.desktop.wm.preferences button-layout 'close,maximize,minimize:'
to move close option to left now to install mac like dock type sudo apt install plank then type plank --preferences in terminal then select
theme to gtk+ position to bottom  allignment to center then in behaviour tab disable hide dock and remove all app icons in dock by right
click and appearance tab enable icon zoom to add apps to the new dock open menu type the app and drag it if you want to remove it drag
of the dock its easy to use to add launch pad open files now we ramed to finder then go to /home/username/.iconthemes drag and drop the
launchpad. then open settings now preferences go to font selection change all fonts default desktop monospace document window title to
lucida grande regular with 9 as size. then go to preferences startup apps select + choose add application select plank enable it.
for the genie effect go to system preferences extensions in download tab type magic lamp effect select download then go to system 
preferences extensions select magic lamp select add for genie effect then genie effect is enabled. now you have macos like ui the old one
for wallpaper use aurorablue.jpg file its found in the unzipped folder of the .zip download
